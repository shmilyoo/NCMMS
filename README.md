# NCMMS

Network centralized monitoring and management system

## requirement

.Net framework 4.0

## 不再维护，no longer maintained since 2012

主要实现了网络拓扑的自动发现，自动绘制（网格、力导向），端口监视，流量监视，批量连通性检测等
